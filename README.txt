Переменные:
httpClient - экземпляр клиента для отправки HTTP-запросов.
requestLimit - максимальное количество запросов в указанный интервал времени.
timeIntervalMillis - интервал времени в миллисекундах.
requestTimestamps - очередь для хранения времени запросов.
requestCount - счетчик текущего количества запросов.

Метод CrptApi (конструктор):
Инициализирует клиент HTTP, устанавливает лимит запросов и интервал времени.
Также инициализирует очередь для хранения времени запросов и счетчик запросов.

Метод createDocument:
Вызывает метод toJson для создания json-строки из объекта "document".
Проверяет ревышение лимита запросов, после чего добавляет текущий запрос в очередь и увеличивает счётчик.
Создание и отправка HTTP запроса

Метод toJson:
Преобразуют объект document в строку json.

Класс Document представляет структуру документа, который будет отправлен на сервер
